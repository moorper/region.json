# region.json

* 试图完善一个完整且持续更新的中国行政区域划分数据
* 数据基于政府行政区域划分制作，数据基本遵照省、市、区、街道，依次划分
* 但并不完全依照省、市、区、街道，制作，比如广东省-东莞市不存在区一级划分，所以结构上就直接为省，市，街道（或者称为区也行），基本思路就是依照从大到小排列
